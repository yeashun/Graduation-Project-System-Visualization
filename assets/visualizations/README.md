# 可视化实验图片目录说明

将论文导出的图片放入以下目录后，系统会自动读取：

- `assets/visualizations/gradcam/baseline.png`
- `assets/visualizations/gradcam/ours.png`
- `assets/visualizations/tsne/baseline.png`
- `assets/visualizations/tsne/ours.png`
- `assets/visualizations/multiscale/baseline.png`
- `assets/visualizations/multiscale/ours.png`

你也可以把文件换成 `.jpg`，然后同步修改 `experiment_data.py` 中对应路径。
